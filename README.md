# Simple C Terminal User Interface
A simple and convenient library for terminal user interface.

## Use
Just include these single-header file `sctui.h` and `skb.h`.
And define their `IMPL` macros to get the definitions for compiling.
- `#include "sctui.h"`: `SCTUI_IMPL`
- `#include "skb.h"`: `SKB_IMPL`

## skb.h
A simple header only library to handle key press.
See `skb.h` to use.
